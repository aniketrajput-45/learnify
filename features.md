# AlgoQuest: Features Overview

AlgoQuest is a gamified Data Structures and Algorithms (DSA) learning platform that transforms technical practice into an immersive RPG experience.

## 🎮 Core Gameplay Mechanics
- **Battle System:** Solve coding problems to "attack" monsters. Correct solutions deal damage, while errors or inefficient code result in player health loss.
- **Dynamic Monsters:** Each level features unique monsters with varying HP and difficulty tiers tied to specific DSA topics.
- **XP & Leveling:** Earn XP for every successful battle. Progress through character levels to unlock new regions of the map.
- **Skill Tree:** A non-linear progression system where players unlock topics (Arrays, Strings, Linked Lists, etc.) in a logical learning sequence.

## 🤖 AI-Powered Mentorship
- **Sensei Hints:** Integrated with **Google Gemini AI**. Stuck on a problem? The "Sensei" analyzes your specific code and error logs to provide strategic hints without spoiling the final answer.
- **Context-Aware Feedback:** The AI understands the problem constraints and your current progress to give tailored advice.

## 🛠️ Advanced Coding Environment
- **Monaco Editor:** Uses the same engine as VS Code, providing syntax highlighting, IntelliSense, and a professional-grade coding experience.
- **Multi-Language Support:** Reference syntax for **C++, Java, Python, and JavaScript** is available for every major topic.
- **Instant Validation:** High-speed, client-side test case execution to provide immediate feedback on code correctness.

## 📚 Knowledge Hub
- **Interactive Library:** A centralized repository explaining core DSA concepts, time complexities, and real-world use cases.
- **Visual Learning:** Clean, structured data presentation for better retention of abstract concepts.

## 🏆 Social & Personalization
- **Global Leaderboard:** Compete with other players globally to see who has the most XP and mastery.
- **User Profiles:** Track your personal stats, including total battles won, accuracy, and current level.
- **Theme Support:** Fully responsive Dark and Light modes for comfortable coding at any time.
- **Fluid Animations:** Powered by **Framer Motion** for a high-polish feel, including damage numbers, combat slashes, and smooth transitions.
